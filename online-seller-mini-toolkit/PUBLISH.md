# Publish The Static Site

This folder is a static site. It can be hosted on GitHub Pages, Netlify, Cloudflare Pages, Vercel, or any static hosting service.

## Before Publishing

Replace placeholders in:

- `index.html`
- `sample.html`
- `demo.html`

Find:

- `PAID_LINK_HERE`
- `FREE_SAMPLE_LINK_HERE`

Replace with your paid product link and free sample link after uploading them to Payhip, Ko-fi, Gumroad, or another checkout.

## Files To Publish

Publish the whole `public-site` folder:

- `index.html`
- `sample.html`
- `demo.html`
- `crypto.html`
- `articles/`
- `styles.css`
- `assets/`
- `robots.txt`
- `sitemap.xml`
- `site.webmanifest`

## Suggested Flow

1. Upload the paid bundle.
2. Upload the free sample.
3. Replace the links in the site files.
4. Publish the site.
5. Put the site link in your social profiles.
6. Use `../distribution/7-day-distribution-plan.md` for the first week of posts.
7. Use `../distribution/14-day-content-calendar.md` for ongoing low-interaction posts.

Before publishing, run `../check-launch.ps1` after building the site.
