## 1. Product Question

Hi, thanks for your interest. This item includes ___. The size/color/version is ___. If you want, I can also send an extra photo before you order.

Chinese note:

Use this when a buyer asks what is included, whether an item is available, or what version they will receive.

## 2. Still Available

Hi, yes, this item is still available. You can order directly through the listing. If you have a question about size, color, delivery, or compatibility, send me a quick message before ordering.

## 3. Shipping Delay

Hi, thanks for your patience. Your order is still on the way. The latest tracking update is ___. I will keep watching it and update you if anything changes.

Chinese note:

Stay calm and specific. Do not promise a delivery date unless the platform or carrier confirms it.

## 4. Refund Request

Hi, I am sorry this did not work out. Please send your order number and a short note about the issue. I will check the order details and follow the platform policy.

## 5. Missing File Or Download

Hi, thanks for letting me know. Please try downloading the file again from your order page. If it still fails, send a screenshot of the error and I will help check the issue.

## 6. Before Purchase Clarification

Hi, this product is a digital download. You will receive ___. It does not include ___. Please check the description before ordering so you know exactly what is included.

## 7. Review Follow-Up

Hi, thank you for your order. If everything worked well, a short review would really help. If something is wrong, please message me first and I will check it.

## 8. Custom Request Boundary

Hi, thanks for asking. This listing includes the files described on the product page. Custom work is not included in this purchase, but I can let you know if I add a version like that later.
